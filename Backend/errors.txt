2xx: Respostas de Sucesso
    Indica que a requisição foi recebida, entendida e aceita com sucesso.

    200 OK: A requisição foi bem-sucedida. Este é o código mais comum.

    201 Created: A requisição foi bem-sucedida e um novo recurso foi criado. Ideal para requisições POST (como um cadastro de usuário).

    202 Accepted: A requisição foi aceita para processamento, mas o processamento não foi concluído.

    204 No Content: O servidor processou a requisição com sucesso, mas não há conteúdo para retornar.

    206 Partial Content: O servidor está enviando apenas uma parte do recurso, conforme solicitado pelo cliente.

3xx: Redirecionamentos
    Indica que o cliente precisa tomar uma ação adicional para completar a requisição.

    301 Moved Permanently: O recurso foi movido permanentemente para uma nova URL.

    302 Found: O recurso foi temporariamente encontrado em uma URL diferente.

    304 Not Modified: O recurso não foi modificado desde a última vez que o cliente o solicitou (usado para caching).

4xx: Erros do Cliente
    Indica que a requisição contém sintaxe inválida, não pode ser entendida ou não pode ser cumprida pelo servidor devido a um erro do cliente.

    400 Bad Request: A requisição não pôde ser entendida pelo servidor (ex: dados de formulário inválidos). Este é o código que você está usando corretamente para validações.

    401 Unauthorized: O cliente não está autenticado e precisa fornecer credenciais para acessar o recurso.

    403 Forbidden: O cliente está autenticado, mas não tem permissão para acessar o recurso.

    404 Not Found: O servidor não conseguiu encontrar o recurso solicitado.

    405 Method Not Allowed: O método HTTP usado (GET, POST, etc.) não é permitido para o recurso.

    408 Request Timeout: O servidor não recebeu a requisição completa a tempo.

    409 Conflict: A requisição não pôde ser concluída por um conflito (ex: tentar criar um recurso que já existe). Este é o código ideal para o seu caso de "e-mail já cadastrado".

    410 Gone: O recurso solicitado foi removido permanentemente.

5xx: Erros do Servidor
    Indica que o servidor falhou ao processar uma requisição aparentemente válida.

    500 Internal Server Error: Um erro genérico no servidor. Use quando não houver outro código mais específico.

    501 Not Implemented: O servidor não suporta a funcionalidade necessária para atender à requisição.

    502 Bad Gateway: O servidor, atuando como um gateway ou proxy, recebeu uma resposta inválida do servidor upstream.

    503 Service Unavailable: O servidor está temporariamente sobrecarregado ou em manutenção.

    504 Gateway Timeout: O servidor, atuando como um gateway, não recebeu uma resposta a tempo de outro servidor.
